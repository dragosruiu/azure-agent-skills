# identity-and-access/

Skills for authenticating workloads, granting access, and managing
secrets — without long-lived credentials.

## In scope

- Managed identity (system- and user-assigned)
- Federated identity credentials (GitHub Actions OIDC, etc.)
- Azure RBAC (built-in roles, custom roles, scoping)
- Key Vault (secrets, keys, certificates) and Key Vault references
- Microsoft Entra ID app registrations as workload identity

## Out of scope (handled elsewhere)

- B2C / consumer identity → not yet covered
- On-behalf-of and delegated user flows for end-user apps → not yet covered
- Networking-level identity (private endpoints) → see `networking/`
